ROX платформа имеет отличный набор аналитических инструментов, которые позволяют моментальную маршрутизацию на все мировые биржи (особенно на крупные биржи США - NASDAQ, AMEX, NYSE). Построена торговая платформа на технологии order management system (OMS).

Как выбрать?

SDG_ Layout для реального счета (файл с настройками)

1. Установите такие настройки (в Панели управления):
• Язык и региональные стандарты -> Форматы -> формат: Английский (США)
• Язык клавиатуры по умолчанию: английский
• Имя ПК: использовать только англ. буквы и цифры
После этого необходимо перезагрузить ПК.

2. Скачайте и установите программу. Используйте при установке демо-дистрибутив для демо-счета и реал-дистрибутив для реального счета.
Установку необходимо производить из аккаунта с правами администратора.

3. Создайте ярлык для запуска программы. Для этого зайдите в папку C:\ROX и найдите файл ROX.exe или ROXdemo.exe (тип файла “Приложение”). Правой кнопкой мышки – “Отправить” – “Рабочий стол (создать ярлык)”. В свойствах ярлыка поставьте галочку "от имени администратора".

3. Установите наш лейаут (файл с настройками). Как это сделать, показано на нашем видео. Используйте демо-лейаут для демо-счета и реал-лейаут для реального счета. По умолчанию програма устанавливается в папку C:\ROX

4. Введите логин и пароль, выданные Вам в письме.

Если программа предложит обновление, необходимо согласиться обновить версию.
